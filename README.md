# Signal Scope

## Quick start

- Install: `npm install`
- Dev server: `npm run dev` (Vite; defaults to http://localhost:5173)
- Build: `npm run build`
- Preview production build: `npm run preview`

## Live demo

- https://signal-scope-rho.vercel.app/

## Notes

- Sample data lives in `public/data.json` and is fetched by the app at those paths.
- Tech stack: React + TypeScript + Vite + Tailwind.
