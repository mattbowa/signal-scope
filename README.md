Signal Scope
============

Quick start
-----------
- Install: `npm install`
- Dev server: `npm run dev` (Vite; defaults to http://localhost:5173)
- Build: `npm run build`
- Preview production build: `npm run preview`

Notes
-----
- Sample data lives in `public/data.json` (and `public/down.json` for the error-state demo) and is fetched by the app at those paths.
- Tech stack: React + TypeScript + Vite + Tailwind.
